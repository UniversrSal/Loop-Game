# Loop

---

##  Overview
Loop is a solo-developed indie horror experience inspired by the quiet unease of walking home ai night, You guide Penny through familiar routines — checking the mail, taking the elevator — unaware that something is slightly…off.


---

## Programs
**Tools / Software**
- Unity
- Autodesk Maya
- Substance Painter
- Photoshop / Procreate
- Visual Studio / VS Code

**Languages**
- C# (Unity)
  

**Other**
- Git / GitHub

---


**Rendering & Visual Processing:**
- Unity URP / HDRP pipelines
- Shader Graph / HLSL programming
- Post-processing stack (bloom, color grading, motion blur)
- Real-time performance analysis

##  Gameplay / Demo 
**Game Loop Summary**

- Player explores → finish objective → go to next scene → repeat
